### 求一个字符串的最长不重复子串

#### 取所有的极长不重复子串中的最大的子串
``` java
public static int Fun(String str){
        if (str.length() == 1) {
            return 1;
        }
        int start = 0;
        int end = 0;
        int index = -1;
        int ans = -1;
        for (end = 1; end < str.length(); end++) {
            index = str.substring(start, end).indexOf(str.charAt(end));
            if (index >= 0) {
                ans = Math.max(ans,end - start);
                start += index + 1;
            }
        }
        return Math.max(ans,end-start);
    }
```

#### 上面解法的优化
``` java
 public static String longestUniqueSubstring(String str){
        // set记录所有已出现过的字符
        HashSet<Object> set = new HashSet<>();
        int start=0,end=0;
        int length = str.length();
        // rp为右指针
        int rp = -1;
        for(int i=0;i<length;i++){
            if( i>0 ) set.remove(str.charAt(i-1));
            while ( rp+1<length && !set.contains(str.charAt(rp+1)) ){
                set.add(str.charAt(rp+1));
                rp+=1;
            }
            // 找到一个极长的无重复子串
            if( rp-i > end-start ){
                end = rp;
                start = i;
            }
        }
        return str.substring(start,end+1);
    }
```
